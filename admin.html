Note : L'API Instagram Basic Display a été dépréciée par Meta en septembre 2024. Le système ci-dessous utilise l'Instagram Graph API (gratuite), qui nécessite un compte Instagram Business ou Creator lié à une Page Facebook. Documentation Meta →
Guide — Obtenir ton Access Token en 3 étapes
1
Crée une application Meta : Va sur developers.facebook.com/apps → "Créer une application" → choisir "Consommateur" ou "Business" → ajouter le produit Instagram Graph API.
2
Convertis ton compte Instagram : Paramètres Instagram → Compte → Passer en compte Professionnel (Creator ou Business). Lie-le à une Page Facebook. Puis dans Meta for Developers → Instagram → ajouter ton compte comme compte de test.
3
Génère un token longue durée : Dans l'Explorateur API Graph de Meta → sélectionne ton app → génère un token avec les permissions instagram_basic, pages_show_list, instagram_manage_insights. Le token longue durée dure 60 jours (renouvelable).

2Copie-les dans le dossier digitalscho\ sur ton PC (remplace les anciens).
